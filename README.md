Simple Password Generator
======

Required:
* Python 2.7
* PyQt4

------

Launch:
* Linux: python passgen.pyw or ./passgen.pyw
* Windows: double click on dist\PassGen2.exe or on passgen.pyw

------

Build:
* Linux: No need
* Windows: launch build.cmd
